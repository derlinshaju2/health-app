📱 Mobile Responsive Test

Test the Diet & Nutrition page mobile responsiveness

🔍 How to Test:

  1. Open Browser DevTools: Press F12 or right-click → Inspect
  2. Enable Device Mode: Click the device icon (Ctrl+Shift+M)
  3. Test Different Devices: iPhone SE, iPhone 12, Samsung Galaxy, etc.
  4. Check for Horizontal Scroll: Try scrolling sideways - there should be NO horizontal scroll
  5. Test Portrait & Landscape: Rotate the device to test both orientations
