DEPLOYMENT TROUBLESHOOTING:
============================

If deployment fails:

1. Check "public" folder is selected (not root directory)

2. Make sure all files are in the public folder

3. Clear Netlify cache:
   - Site settings → Build & deploy
   - Click "Clear cache and retry deploy"

4. If using Git integration, disconnect and reconnect:
   - Site settings → Site configuration
   - Click "Change repository"
   - Reconnect GitHub with correct settings
